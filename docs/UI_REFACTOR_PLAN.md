# 博客UI和Pages重构规划

## 🎨 概览

本文档规划博客的UI界面和页面重构,打造一个现代化、专业的技术博客。

---

## 📋 目标

1. **现代化设计**: 采用最新的Web设计趋势
2. **用户体验优化**: 提升阅读体验和导航便捷性  
3. **功能丰富**: 添加更多实用页面和功能
4. **性能优化**: 快速加载和流畅交互
5. **响应式设计**: 完美适配各种设备

---

## 🎨 UI设计方向

### 整体风格

- **设计语言**: 简洁现代,扁平化设计
- **配色方案**:
  - 主色: 蓝色系 (#2196F3 / #1976D2)
  - 辅色: 紫色系 (#9C27B0)
  - 强调: 橙色系 (#FF9800)
  - 背景: 白色/浅灰
  - 深色模式: 暗蓝灰 (#1a1a2e / #16213e)

### 字体方案

**中文**:

- 标题: `Noto Sans SC / 思源黑体`
- 正文: `Noto Serif SC / 思源宋体`
- 代码: `JetBrains Mono / Fira Code`

**英文**:

- 标题: `Inter / Montserrat`
- 正文: `Lora / Merriweather`
- 代码: `JetBrains Mono / Fira Code`

### 组件设计

1. **导航栏**
   - 固定顶部导航
   - Logo + 主菜单
   - 搜索框集成
   - 深色模式切换
   - 移动端汉堡菜单

2. **首页Hero区域**
   - 大标题 + Slogan
   - 动态背景(粒子效果/渐变)
   - 社交链接
   - 快速导航卡片

3. **文章卡片**
   - 缩略图 + 标题
   - 摘要 + 阅读时间
   - 分类和标签标记
   - Hover效果(阴影/放大)

4. **侧边栏组件**
   - 个人简介卡片
   - 最新文章列表
   - 分类云
   - 标签云
   - 归档日历

5. **文章页面**
   - 清晰的排版
   - 目录导航(TOC)
   - 代码高亮
   - 数学公式支持
   - 图片灯箱效果
   - 文章导航(上/下一篇)

6. **页脚**
   - 多栏布局
   - 站点信息
   - 友情链接
   - 社交媒体
   -备案信息

┌─────────────────────────────────────┐

├─────────────────────────────────────┤
│         Hero 区域                    │
│    - 大标题/Slogan                  │
│    - 动态背景                        │
│    - CTA按钮                        │
├─────────────────────────────────────┤
│         特色文章区域                 │
│    - 3个精选文章卡片                │

│         最新文章列表                 │

## 🛠️ 技术实现

### Hexo主题选择

**推荐主题**:

1. **Butterfly** (当前) - 功能丰富
2. **NexT** - 简洁优雅
3. **Fluid** - 现代设计
4. **Icarus** - 卡片式布局
5. **Matery** - Material Design

**或自定义主题**

### 必备插件

```bash
# 搜索
npm install hexo-generator-search

# RSS
npm install hexo-generator-feed

# Sitemap
npm install hexo-generator-sitemap

# 压缩
npm install hexo-all-minifier

# 图片处理
npm install hexo-image-sizes

# 阅读统计
npm install hexo-word-count

# 相关文章
npm install hexo-related-posts
```

### 自定义样式

创建 `source/css/custom.css`:

```css
/* 自定义配色 */
:root {
  --primary-color: #2196F3;
  --secondary-color: #9C27B0;
  --accent-color: #FF9800;
  --text-color: #333;
  --bg-color: #fff;
}

/* 深色模式 */
[data-theme="dark"] {
  --primary-color: #42A5F5;
  --text-color: #E0E0E0;
  --bg-color: #1a1a2e;
}

/* 自定义字体 */
body {
  font-family: 'Noto Serif SC', serif;
}

h1, h2, h3, h4, h5, h6 {
  font-family: 'Noto Sans SC', sans-serif;
}

code, pre {
  font-family: 'JetBrains Mono', monospace;
}
```

  /* 两列布局 */

/* 桌面设备 */
@media (min-width: 1025px) {
  /* 三列布局 */
