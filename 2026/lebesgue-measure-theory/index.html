实分析中的勒贝格测度理论

勒贝格测度是现代分析学的基石，为积分理论提供了坚实的基础。本文将系统介绍测度论的核心概念。

1. 测度的基本概念

1.1 σ-代数

设 XXX 为非空集合，F⊆P(X)\mathcal{F} \subseteq \mathcal{P}(X)F⊆P(X) 称为 σ-代数，若满足：

  1. X∈FX \in \mathcal{F}X∈F
  2. 补封闭：若 A∈FA \in \mathcal{F}A∈F，则 Ac∈FA^c \in \mathcal{F}Ac∈F
  3. 可数并封闭：若 {An}n=1∞⊆F\{A_n\}_{n=1}^\infty \subseteq \mathcal{F}{An​}n=1∞​⊆F，则 ⋃n=1∞An∈F\bigcup_{n=1}^\infty A_n \in \mathcal{F}⋃n=1∞​An​∈F

(X,F)(X, \mathcal{F})(X,F) 称为可测空间。

1.2 测度的定义

设 (X,F)(X, \mathcal{F})(X,F) 为可测空间，函数 μ:F→[0,∞]\mu: \mathcal{F} \to [0, \infty]μ:F→[0,∞] 称为测度，若满足：

  1. 非负性：μ(A)≥0\mu(A) \geq 0μ(A)≥0，∀A∈F\forall A \in \mathcal{F}∀A∈F
  2. 空集测度为零：μ(∅)=0\mu(\emptyset) = 0μ(∅)=0
  3. 可数可加性：若 {An}n=1∞\{A_n\}_{n=1}^\infty{An​}n=1∞​ 两两不相交，则

μ(⋃n=1∞An)=∑n=1∞μ(An)\mu\left(\bigcup_{n=1}^\infty A_n\right) = \sum_{n=1}^\infty \mu(A_n) μ(n=1⋃∞​An​)=n=1∑∞​μ(An​)

三元组 (X,F,μ)(X, \mathcal{F}, \mu)(X,F,μ) 称为测度空间。

2. 勒贝格测度的构造

2.1 外测度方法

外测度定义：对于 E⊆RE \subseteq \mathbb{R}E⊆R，定义

μ∗(E)=inf⁡{∑n=1∞ℓ(In):E⊆⋃n=1∞In, In 为开区间}\mu^*(E) = \inf \left\{ \sum_{n=1}^\infty \ell(I_n) : E \subseteq \bigcup_{n=1}^\infty I_n, \, I_n \text{ 为开区间} \right\} μ∗(E)=inf{n=1∑∞​ℓ(In​):E⊆n=1⋃∞​In​,In​ 为开区间}

其中 ℓ((a,b))=b−a\ell((a, b)) = b - aℓ((a,b))=b−a 为区间长度。

外测度性质：

  1. μ∗(∅)=0\mu^*(\emptyset) = 0μ∗(∅)=0
  2. 单调性：A⊆B⇒μ∗(A)≤μ∗(B)A \subseteq B \Rightarrow \mu^*(A) \leq \mu^*(B)A⊆B⇒μ∗(A)≤μ∗(B)
  3. 次可加性：μ∗(⋃n=1∞An)≤∑n=1∞μ∗(An)\mu^*\left(\bigcup_{n=1}^\infty A_n\right) \leq \sum_{n=1}^\infty \mu^*(A_n)μ∗(⋃n=1∞​An​)≤∑n=1∞​μ∗(An​)

2.2 Carathéodory 可测性

集合 EEE 称为 Carathéodory 可测的（或勒贝格可测的），若对任意 A⊆RA \subseteq \mathbb{R}A⊆R：

μ∗(A)=μ∗(A∩E)+μ∗(A∩Ec)\mu^*(A) = \mu^*(A \cap E) + \mu^*(A \cap E^c) μ∗(A)=μ∗(A∩E)+μ∗(A∩Ec)

定理：所有勒贝格可测集构成 σ-代数 L\mathcal{L}L，外测度限制在 L\mathcal{L}L 上是完备测度。

2.3 Borel 集与勒贝格集

开集⊆B(R)⊆L\text{开集} \subseteq \mathcal{B}(\mathbb{R}) \subseteq \mathcal{L} 开集⊆B(R)⊆L

其中 B(R)\mathcal{B}(\mathbb{R})B(R) 是由开集生成的 σ-代数（Borel σ-代数），包含关系严格。

3. Cantor 集与测度零集

3.1 Cantor 集的构造

从 [0,1][0, 1][0,1] 开始，迭代地删除每个区间的中间 13\frac{1}{3}31​：

C0=[0,1]C_0 = [0, 1] C0​=[0,1]

C1=[0,13]∪[23,1]C_1 = \left[0, \frac{1}{3}\right] \cup \left[\frac{2}{3}, 1\right] C1​=[0,31​]∪[32​,1]

C2=[0,19]∪[29,13]∪[23,79]∪[89,1]C_2 = \left[0, \frac{1}{9}\right] \cup \left[\frac{2}{9}, \frac{1}{3}\right] \cup \left[\frac{2}{3}, \frac{7}{9}\right] \cup \left[\frac{8}{9}, 1\right] C2​=[0,91​]∪[92​,31​]∪[32​,97​]∪[98​,1]

Cantor 集：C=⋂n=0∞CnC = \bigcap_{n=0}^\infty C_nC=⋂n=0∞​Cn​

3.2 Cantor 集的性质

  1. 闭集：CCC 是闭集
  2. 无处稠密：CCC 不包含任何开区间
  3. 不可数：CCC 与 [0,1][0, 1][0,1] 等势（可用三进制表示证明）
  4. 测度为零：

μ(C)=lim⁡n→∞μ(Cn)=lim⁡n→∞(23)n=0\mu(C) = \lim_{n \to \infty} \mu(C_n) = \lim_{n \to \infty} \left(\frac{2}{3}\right)^n = 0 μ(C)=n→∞lim​μ(Cn​)=n→∞lim​(32​)n=0

Cantor 集展示了"不可数但测度为零"的奇特现象。

4. 勒贝格积分

4.1 简单函数

设 (X,F,μ)(X, \mathcal{F}, \mu)(X,F,μ) 为测度空间，简单函数是形如

s(x)=∑i=1naiχAi(x)s(x) = \sum_{i=1}^n a_i \chi_{A_i}(x) s(x)=i=1∑n​ai​χAi​​(x)

的函数，其中 ai≥0a_i \geq 0ai​≥0，Ai∈FA_i \in \mathcal{F}Ai​∈F，χAi\chi_{A_i}χAi​​ 是特征函数。

简单函数的积分：

∫Xs dμ=∑i=1naiμ(Ai)\int_X s \, d\mu = \sum_{i=1}^n a_i \mu(A_i) ∫X​sdμ=i=1∑n​ai​μ(Ai​)

4.2 一般非负函数的积分

对于可测函数 f:X→[0,∞]f: X \to [0, \infty]f:X→[0,∞]：

∫Xf dμ=sup⁡{∫Xs dμ:0≤s≤f, s 为简单函数}\int_X f \, d\mu = \sup \left\{ \int_X s \, d\mu : 0 \leq s \leq f, \, s \text{ 为简单函数} \right\} ∫X​fdμ=sup{∫X​sdμ:0≤s≤f,s 为简单函数}

4.3 重要收敛定理

单调收敛定理 (MCT)

若 0≤f1≤f2≤⋯0 \leq f_1 \leq f_2 \leq \cdots0≤f1​≤f2​≤⋯ 且 fn→ff_n \to ffn​→f 点点收敛，则

lim⁡n→∞∫Xfn dμ=∫Xf dμ\lim_{n \to \infty} \int_X f_n \, d\mu = \int_X f \, d\mu n→∞lim​∫X​fn​dμ=∫X​fdμ

控制收敛定理 (DCT)

若 fn→ff_n \to ffn​→f 点点收敛，且存在可积函数 ggg 使得 ∣fn∣≤g|f_n| \leq g∣fn​∣≤g，则

lim⁡n→∞∫Xfn dμ=∫Xf dμ\lim_{n \to \infty} \int_X f_n \, d\mu = \int_X f \, d\mu n→∞lim​∫X​fn​dμ=∫X​fdμ

5. L^p 空间的完备性

5.1 定义

Lp(Ω)={f:Ω→R | ∥f∥p=(∫Ω∣f∣p dμ)1/p<∞}L^p(\Omega) = \left\{ f : \Omega \to \mathbb{R} \,\middle|\, \|f\|_p = \left(\int_\Omega |f|^p \, d\mu\right)^{1/p} < \infty \right\} Lp(Ω)={f:Ω→R​∥f∥p​=(∫Ω​∣f∣pdμ)1/p<∞}

5.2 Riesz-Fischer 定理

定理：对于 1≤p≤∞1 \leq p \leq \infty1≤p≤∞，Lp(Ω)L^p(\Omega)Lp(Ω) 是 Banach 空间。

证明思路（1≤p<∞1 \leq p < \infty1≤p<∞）：

  1. 取 Cauchy 列 {fn}\{f_n\}{fn​}
  2. 构造快速收敛子列
  3. 证明点点收敛到某 fff
  4. 利用 Fatou 引理证明 f∈Lpf \in L^pf∈Lp
  5. 证明 ∥fn−f∥p→0\|f_n - f\|_p \to 0∥fn​−f∥p​→0
