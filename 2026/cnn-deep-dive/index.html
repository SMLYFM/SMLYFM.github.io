卷积神经网络 (CNN) 原理详解

发表于2026-01-25|更新于2026-01-25|计算机深度学习
|总字数:2.1k|阅读时长:9分钟|浏览量:

引言

卷积神经网络（Convolutional Neural Network, CNN）是深度学习在计算机视觉领域最成功的模型之一。从图像分类到目标检测，从语义分割到图像生成，CNN 无处不在。

1. 为什么需要 CNN

1.1 全连接网络的问题

对于 224×224224 \times 224224×224 的 RGB 图像，展平后有 224×224×3=150,528224 \times 224 \times 3 = 150,528224×224×3=150,528 个输入。如果第一个隐藏层有 1000 个神经元，则需要约 1.5 亿个参数！

问题：

  1. 参数爆炸：参数过多导致过拟合和计算困难
  2. 忽略空间结构：展平操作丢失了像素的空间关系
  3. 缺乏平移不变性：同一物体在不同位置需要重新学习

1.2 CNN 的设计直觉

CNN 的核心思想来自视觉神经科学：

  • 局部感受野：神经元只对视野中的局部区域响应
  • 权值共享：相同的特征检测器用于整个图像
  • 层次化表示：从边缘到形状到物体的逐层抽象

2. 卷积运算

2.1 数学定义

对于输入 xxx 和卷积核 www，卷积运算（实际上是互相关）：

(x∗w)(i,j)=∑m∑nx(i+m,j+n)⋅w(m,n)(x * w)(i, j) = \sum_{m} \sum_{n} x(i+m, j+n) \cdot w(m, n) (x∗w)(i,j)=m∑​n∑​x(i+m,j+n)⋅w(m,n)

2.2 卷积层参数

输入：H×W×CinH \times W \times C_{in}H×W×Cin​（高度 × 宽度 × 输入通道）

卷积核：K×K×Cin×CoutK \times K \times C_{in} \times C_{out}K×K×Cin​×Cout​

输出大小：

Hout=H−K+2PS+1H_{out} = \frac{H - K + 2P}{S} + 1 Hout​=SH−K+2P​+1

其中：

  • KKK：卷积核大小
  • PPP：填充（padding）
  • SSS：步幅（stride）

对于 RGB 图像（3 通道），每个卷积核也有 3 层，分别与对应通道卷积后求和：

y=∑c=1Cinxc∗wc+by = \sum_{c=1}^{C_{in}} x_c * w_c + b y=c=1∑Cin​​xc​∗wc​+b

2.4 1×1 卷积

1×11 \times 11×1 卷积不改变空间尺寸，但可以：

  • 改变通道数：降维或升维
  • 增加非线性：每个卷积后跟激活函数
  • 跨通道信息交互

3. 池化层

3.1 最大池化

取窗口内的最大值：

作用：

  • 降低空间分辨率
  • 提供一定的平移不变性
  • 减少计算量

对整个特征图求平均，常用于分类网络的最后：

优点：无参数，减少过拟合。

4. 激活函数

4.1 ReLU

ReLU(x)=max⁡(0,x)\text{ReLU}(x) = \max(0, x) ReLU(x)=max(0,x)

优点：

  • 计算简单
  • 缓解梯度消失
  • 稀疏激活

其中 Φ\PhiΦ 是标准正态分布的 CDF。在 Transformer 中广泛使用。

5. 经典 CNN 架构

5.1 LeNet-5 (1998)

5.2 AlexNet (2012)

ImageNet 竞赛的突破性工作，深度学习复兴的标志。

创新点：

  • 使用 ReLU 激活函数
  • 使用 Dropout 防止过拟合
  • 数据增强（随机裁剪、翻转）
  • GPU 训练

5.3 VGGNet (2014)

核心思想：使用小卷积核（3×33 \times 33×3）堆叠代替大卷积核。

两个 3×33 \times 33×3 卷积 = 一个 5×55 \times 55×5 感受野，但参数更少：

2×(32)=18<52=252 \times (3^2) = 18 < 5^2 = 25 2×(32)=18<52=25

5.4 GoogLeNet/Inception (2014)

Inception 模块：并行使用不同大小的卷积核，然后拼接。

使用 1×11 \times 11×1 卷积降维减少计算量。

5.5 ResNet (2015)

核心创新：残差连接（Skip Connection）

y=F(x)+x\mathbf{y} = \mathcal{F}(\mathbf{x}) + \mathbf{x} y=F(x)+x

网络学习残差 F(x)=y−x\mathcal{F}(\mathbf{x}) = \mathbf{y} - \mathbf{x}F(x)=y−x，而不是直接学习 y\mathbf{y}y。

残差块：

为什么有效：

  • 梯度可以直接通过跳跃连接回传
  • 允许训练非常深的网络（152+ 层）
  • 隐式集成了不同深度的网络

6.1 定义

对于 mini-batch 中的每个特征：

yi=γx^i+βy_i = \gamma \hat{x}_i + \beta yi​=γx^i​+β

其中 γ,β\gamma, \betaγ,β 是可学习参数。

6.2 作用

  1. 加速训练：允许更大的学习率
  2. 正则化效果：mini-batch 统计带来噪声
  3. 减少内部协变量偏移

8. CNN 的理论理解

8.1 感受野

第 LLL 层的感受野大小：

RFL=RFL−1+(KL−1)×∏i=1L−1SiRF_L = RF_{L-1} + (K_L - 1) \times \prod_{i=1}^{L-1} S_i RFL​=RFL−1​+(KL​−1)×i=1∏L−1​Si​

深层神经元"看到"的输入区域更大。

8.2 特征可视化

浅层学习：边缘、颜色、纹理
中层学习：角点、形状、纹理组合
深层学习：物体部件、语义概念

8.3 平移等变性

卷积具有平移等变性：

Tt(f∗g)=(Ttf)∗gT_\mathbf{t}(f * g) = (T_\mathbf{t} f) * g Tt​(f∗g)=(Tt​f)∗g

输入平移导致输出相同平移。

总结

CNN 的核心设计原则：

  1. 局部连接：减少参数，利用空间局部性
  2. 权值共享：同一特征检测器检测整张图
  3. 层次表示：从低级到高级特征逐层抽象
  4. 池化下采样：降低分辨率，增加感受野
  5. 残差连接：训练更深的网络
