泛函分析基础：Banach 空间与 Hilbert 空间

泛函分析是现代数学的重要分支，为偏微分方程、量子力学等提供理论基础。本文将系统介绍两大核心空间。

1. Banach 空间

1.1 定义

设 XXX 为实（或复）线性空间，若存在函数 ∥⋅∥:X→[0,∞)\|\cdot\|: X \to [0, \infty)∥⋅∥:X→[0,∞) 满足：

  1. 正定性：∥x∥=0⇔x=0\|x\| = 0 \Leftrightarrow x = 0∥x∥=0⇔x=0
  2. 齐次性：∥αx∥=∣α∣∥x∥\|\alpha x\| = |\alpha| \|x\|∥αx∥=∣α∣∥x∥，∀α∈K,x∈X\forall \alpha \in \mathbb{K}, x \in X∀α∈K,x∈X
  3. 三角不等式：∥x+y∥≤∥x∥+∥y∥\|x + y\| \leq \|x\| + \|y\|∥x+y∥≤∥x∥+∥y∥

则称 (X,∥⋅∥)(X, \|\cdot\|)(X,∥⋅∥) 为赋范线性空间。若 (X,∥⋅∥)(X, \|\cdot\|)(X,∥⋅∥) 关于由范数诱导的度量完备，则称其为 Banach 空间。

1.2 典型例子

LpL^pLp 空间

设 (Ω,F,μ)(\Omega, \mathcal{F}, \mu)(Ω,F,μ) 为测度空间，定义

Lp(Ω)={f:Ω→R | ∥f∥p:=(∫Ω∣f∣p dμ)1/p<∞}L^p(\Omega) = \left\{ f: \Omega \to \mathbb{R} \,\middle|\, \|f\|_p := \left( \int_\Omega |f|^p \, d\mu \right)^{1/p} < \infty \right\} Lp(Ω)={f:Ω→R​∥f∥p​:=(∫Ω​∣f∣pdμ)1/p<∞}

其中 1≤p<∞1 \leq p < \infty1≤p<∞。由 Riesz-Fischer 定理，LpL^pLp 空间是完备的。

ℓp\ell^pℓp 序列空间

ℓp={(xn)n=1∞ | ∑n=1∞∣xn∣p<∞}\ell^p = \left\{ (x_n)_{n=1}^\infty \,\middle|\, \sum_{n=1}^\infty |x_n|^p < \infty \right\} ℓp={(xn​)n=1∞​​n=1∑∞​∣xn​∣p<∞}

1.3 重要定理

开映射定理 (Open Mapping Theorem)

设 X,YX, YX,Y 为 Banach 空间，T:X→YT: X \to YT:X→Y 为有界线性算子且为满射，则 TTT 为开映射。

闭图像定理 (Closed Graph Theorem)

设 X,YX, YX,Y 为 Banach 空间，T:X→YT: X \to YT:X→Y 为线性算子。若 TTT 的图像

Γ(T)={(x,Tx):x∈X}⊆X×Y\Gamma(T) = \{(x, Tx) : x \in X\} \subseteq X \times Y Γ(T)={(x,Tx):x∈X}⊆X×Y

在乘积拓扑下是闭集，则 TTT 有界。

2. Hilbert 空间

2.1 定义

设 HHH 为复线性空间，若存在内积 ⟨⋅,⋅⟩:H×H→C\langle \cdot, \cdot \rangle: H \times H \to \mathbb{C}⟨⋅,⋅⟩:H×H→C 满足：

  1. 共轭对称性：⟨x,y⟩=⟨y,x⟩‾\langle x, y \rangle = \overline{\langle y, x \rangle}⟨x,y⟩=⟨y,x⟩​
  2. 第一变元线性：⟨αx+βy,z⟩=α⟨x,z⟩+β⟨y,z⟩\langle \alpha x + \beta y, z \rangle = \alpha \langle x, z \rangle + \beta \langle y, z \rangle⟨αx+βy,z⟩=α⟨x,z⟩+β⟨y,z⟩
  3. 正定性：⟨x,x⟩≥0\langle x, x \rangle \geq 0⟨x,x⟩≥0，等号当且仅当 x=0x = 0x=0

则称 (H,⟨⋅,⋅⟩)(H, \langle \cdot, \cdot \rangle)(H,⟨⋅,⋅⟩) 为内积空间。若 HHH 关于由内积诱导的范数

∥x∥=⟨x,x⟩\|x\| = \sqrt{\langle x, x \rangle} ∥x∥=⟨x,x⟩​

完备，则称 HHH 为 Hilbert 空间。

2.2 重要性质

平行四边形恒等式

∥x+y∥2+∥x−y∥2=2(∥x∥2+∥y∥2)\|x + y\|^2 + \|x - y\|^2 = 2(\|x\|^2 + \|y\|^2) ∥x+y∥2+∥x−y∥2=2(∥x∥2+∥y∥2)

这是 Hilbert 空间的特征性质，Banach 空间不一定满足。

正交分解定理

设 MMM 为 Hilbert 空间 HHH 的闭子空间，则

H=M⊕M⊥H = M \oplus M^\perp H=M⊕M⊥

其中 M⊥={x∈H:⟨x,m⟩=0,∀m∈M}M^\perp = \{x \in H : \langle x, m \rangle = 0, \forall m \in M\}M⊥={x∈H:⟨x,m⟩=0,∀m∈M}。

2.3 典型例子

L2L^2L2 空间

L2(Ω)={f:Ω→C | ∫Ω∣f∣2 dμ<∞}L^2(\Omega) = \left\{ f: \Omega \to \mathbb{C} \,\middle|\, \int_\Omega |f|^2 \, d\mu < \infty \right\} L2(Ω)={f:Ω→C​∫Ω​∣f∣2dμ<∞}

内积定义为 ⟨f,g⟩=∫Ωfg‾ dμ\langle f, g \rangle = \int_\Omega f \overline{g} \, d\mu⟨f,g⟩=∫Ω​fg​dμ。

ℓ2\ell^2ℓ2 序列空间

ℓ2={(xn) | ∑n=1∞∣xn∣2<∞}\ell^2 = \left\{ (x_n) \,\middle|\, \sum_{n=1}^\infty |x_n|^2 < \infty \right\} ℓ2={(xn​)​n=1∑∞​∣xn​∣2<∞}

内积定义为 ⟨x,y⟩=∑n=1∞xnyn‾\langle x, y \rangle = \sum_{n=1}^\infty x_n \overline{y_n}⟨x,y⟩=∑n=1∞​xn​yn​​。

3. Banach 空间与 Hilbert 空间的关系

Hilbert 空间是特殊的 Banach 空间

性质 Banach 空间 Hilbert 空间
范数结构 ✅ ✅
内积结构 ❌ ✅
平行四边形恒等式 不一定 ✅
正交分解 不一定 ✅
