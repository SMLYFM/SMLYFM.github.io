数学猜想：从费马到黎曼

发表于2026-01-25|更新于2026-01-25|数学分析学
|总字数:1.8k|阅读时长:6分钟|浏览量:

引言

数学猜想是推动数学发展的重要动力。一个好的猜想往往能够揭示数学结构的深层联系，激发几代数学家的探索热情。本文将介绍几个著名的数学猜想，探讨它们的历史背景、数学意义和当前状态。

1. 费马大定理

1.1 问题陈述

费马大定理（Fermat’s Last Theorem）：对于任意整数 n>2n > 2n>2，方程

xn+yn=znx^n + y^n = z^n xn+yn=zn

没有正整数解。

费马在 1637 年左右在《算术》一书的页边写下了这个猜想，并声称他有一个"绝妙的证明"，但页边太窄写不下。

1.2 历史进程

年份 数学家 贡献
1753 欧拉 证明 n=3n=3n=3 的情形
1825 迪利克雷、勒让德 证明 n=5n=5n=5 的情形
1839 拉梅 证明 n=7n=7n=7 的情形
1850 库默尔 引入理想数，证明所有正则素数情形
1995 怀尔斯 完成完整证明

1.3 怀尔斯的证明

安德鲁·怀尔斯（Andrew Wiles）的证明基于以下深刻联系：

谷山-志村猜想：每个定义在 Q\mathbb{Q}Q 上的椭圆曲线都是模形式。

证明路径：

  1. 若费马大定理有解 (a,b,c)(a, b, c)(a,b,c)，则可构造椭圆曲线 E:y2=x(x−an)(x+bn)E: y^2 = x(x-a^n)(x+b^n)E:y2=x(x−an)(x+bn)
  2. 弗雷证明这个曲线不是模形式（弗雷曲线）
  3. 怀尔斯证明所有半稳定椭圆曲线都是模形式
  4. 矛盾！费马大定理成立

2. 黎曼假设

2.1 黎曼 zeta 函数

定义：对于 ℜ(s)>1\Re(s) > 1ℜ(s)>1，黎曼 zeta 函数定义为：

ζ(s)=∑n=1∞1ns=∏p 素数11−p−s\zeta(s) = \sum_{n=1}^{\infty} \frac{1}{n^s} = \prod_{p \text{ 素数}} \frac{1}{1 - p^{-s}} ζ(s)=n=1∑∞​ns1​=p 素数∏​1−p−s1​

右侧的欧拉乘积公式揭示了 ζ(s)\zeta(s)ζ(s) 与素数的深刻联系。

2.2 猜想陈述

黎曼假设（Riemann Hypothesis）：ζ(s)\zeta(s)ζ(s) 的所有非平凡零点的实部都等于 12\frac{1}{2}21​。

即所有非平凡零点都位于临界线 ℜ(s)=12\Re(s) = \frac{1}{2}ℜ(s)=21​ 上。

2.3 与素数分布的联系

素数计数函数 π(x)\pi(x)π(x) 与 ζ(s)\zeta(s)ζ(s) 零点的关系：

π(x)=Li(x)−∑ρLi(xρ)+小阶项\pi(x) = \text{Li}(x) - \sum_{\rho} \text{Li}(x^\rho) + \text{小阶项} π(x)=Li(x)−ρ∑​Li(xρ)+小阶项

其中 ρ\rhoρ 遍历 ζ(s)\zeta(s)ζ(s) 的所有非平凡零点。

如果黎曼假设成立，则：

π(x)=Li(x)+O(xlog⁡x)\pi(x) = \text{Li}(x) + O(\sqrt{x} \log x) π(x)=Li(x)+O(x​logx)

这给出了素数分布的最佳误差估计。

2.4 已知结果

  • 已验证前 101310^{13}1013 个零点都在临界线上
  • Hardy (1914)：临界线上有无穷多个零点
  • Selberg (1942)：至少 40%40\%40% 的零点在临界线上
  • Conrey (1989)：至少 40%40\%40% 的零点在临界线上（改进估计）

3. 哥德巴赫猜想

3.1 问题陈述

强哥德巴赫猜想：每个大于 2 的偶数都可以表示为两个素数之和。

例如：4=2+24 = 2+24=2+2，6=3+36 = 3+36=3+3，8=3+58 = 3+58=3+5，10=5+5=3+710 = 5+5 = 3+710=5+5=3+7

弱哥德巴赫猜想：每个大于 5 的奇数都可以表示为三个素数之和。

3.2 进展

  • 陈氏定理 (1966)：每个充分大的偶数都可以表示为一个素数与一个半素数（最多两个素因子的数）之和
  • 弱猜想已解决 (2013)：哈罗德·赫尔夫戈特完整证明了弱哥德巴赫猜想

3.3 数值验证

强猜想已验证到 4×10184 \times 10^{18}4×1018，未发现反例。

4. 孪生素数猜想

4.1 问题陈述

孪生素数猜想：存在无穷多对孪生素数 (p,p+2)(p, p+2)(p,p+2)。

孪生素数例子：(3,5),(5,7),(11,13),(17,19),(29,31),…(3,5), (5,7), (11,13), (17,19), (29,31), \ldots(3,5),(5,7),(11,13),(17,19),(29,31),…

4.2 重大突破

张益唐 (2013)：存在无穷多对素数 (p,q)(p, q)(p,q)，满足 q−p<7×107q - p < 7 \times 10^7q−p<7×107。

这是第一次证明存在有界间隔的无穷多对素数！

Polymath 项目：将上界降到 246。

Maynard-Tao (2014)：将间隔上界降到 600（独立证明）。

目前最佳结果：如果广义黎曼假设成立，上界可降到 6。

5. P vs NP 问题

5.1 复杂性类

  • P：多项式时间内可解决的问题
  • NP：多项式时间内可验证解的问题
  • NP-完全：NP 中最难的问题

5.2 问题陈述

P=NP ?\textbf{P} = \textbf{NP} \text{ ?} P=NP ?

这个问题问的是：如果一个问题的解可以快速验证，它是否也可以快速找到？

5.3 意义

如果 P=NPP = NPP=NP：

  • 所有 NP 问题都有高效算法
  • 密码学的基础将被动摇
  • 许多优化问题将变得"容易"

主流猜测：P≠NPP \neq NPP=NP

6. BSD 猜想

6.1 椭圆曲线

椭圆曲线 EEE 定义为：

E:y2=x3+ax+bE: y^2 = x^3 + ax + b E:y2=x3+ax+b

其有理点构成一个有限生成的阿贝尔群：

E(Q)≅Zr⊕E(Q)torsE(\mathbb{Q}) \cong \mathbb{Z}^r \oplus E(\mathbb{Q})_{\text{tors}} E(Q)≅Zr⊕E(Q)tors​

其中 rrr 称为秩（rank）。

6.2 L-函数

椭圆曲线的 L-函数 L(E,s)L(E, s)L(E,s) 在 s=1s = 1s=1 处的性质与曲线的算术性质密切相关。

6.3 猜想陈述

BSD 猜想：

  1. r=ords=1L(E,s)r = \text{ord}_{s=1} L(E, s)r=ords=1​L(E,s)（秩等于 L-函数在 s=1s=1s=1 处的零点阶数）
  2. 存在精确的公式计算首项系数，涉及 Tate-Shafarevich 群

7. 七个千禧年问题

克雷数学研究所在 2000 年提出七个千禧年问题，每个奖金 100 万美元：

问题 状态
P vs NP 未解决
霍奇猜想 未解决
庞加莱猜想 ✅ 已解决 (佩雷尔曼, 2003)
黎曼假设 未解决
杨-米尔斯存在性与质量间隙 未解决
纳维-斯托克斯方程 未解决
BSD 猜想 未解决

总结

数学猜想体现了人类探索数学真理的勇气和智慧：

  1. 费马大定理：历经 358 年终于被证明，揭示了椭圆曲线与模形式的深刻联系
  2. 黎曼假设：素数分布的终极问题，至今悬而未决
  3. 哥德巴赫猜想：简单易懂却极难证明的加性数论问题
  4. 孪生素数：近年取得重大突破，但完整证明仍遥遥无期
  5. P vs NP：计算复杂性理论的核心问题

这些问题不仅是智力挑战，更是数学发展的灯塔。
